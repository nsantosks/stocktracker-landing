CONCATENATE()

La expresión CONCATENATE() une (concatena) dos o más fragmentos de texto o valores de columnas para formar una única cadena de texto continua.

Es esencial para crear "Nombre Completos" uniendo nombre y apellido, generar IDs únicos combinando fechas y códigos, o construir direcciones completas a partir de calles y números.

Sintaxis de la expresión

CONCATENATE( Texto1, [Texto2], [Texto3]... )

Parámetros:

  • Texto1, Texto2...: Las partes que deseas unir. Pueden ser columnas (ej: [Nombre]), texto fijo entre comillas (ej: " - ") o resultados de otras fórmulas.
Consejo Pro: También puedes usar el operador & para hacer lo mismo de forma más corta.
[Nombre] & " " & [Apellido] es exactamente lo mismo que CONCATENATE([Nombre], " ", [Apellido]).

Ejemplos Prácticos

  • Crear Nombre Completo:
    CONCATENATE([Nombre], " ", [Apellido])
    Resultado: "Juan Pérez" (El espacio en medio es importante para que no queden pegados).
  • Generar un ID único personalizado:
    CONCATENATE("ORD-", [ID_Cliente], "-", TEXT(TODAY(), "ddmmyy"))
    Resultado: "ORD-105-091223"
  • Crear un mensaje dinámico para notificaciones:
    CONCATENATE("Hola ", [Nombre], ", tu pedido #", [ID_Pedido], " ha sido enviado.")
