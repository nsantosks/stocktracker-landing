EXTRACT()

La expresión EXTRACT() de AppSheet permite identificar y extraer partes específicas de un texto basándose en patrones reconocibles.

Es extremadamente útil para procesar datos desordenados. Puedes elegir extraer objetivos como:

  • NUMBERS (Números)
  • EMAILS (Correos electrónicos)
  • PRICES (Precios)
  • DATES (Fechas)
  • PHONENUMBERS (Teléfonos)

Sintaxis de la expresión

EXTRACT( "Objetivo", [Columna o Texto] )

Parámetros:

  • Objetivo: El tipo de dato que deseas buscar (ej: "EMAILS", "NUMBERS").
  • Texto: La columna o cadena de texto donde se buscará.

Ejemplos Prácticos

  • Extraer correos de una nota:
    EXTRACT("EMAILS", "Contactar a soporte@gammaliel.com para ayuda")
    Resultado: Una lista que contiene "soporte@gammaliel.com".
  • Obtener precios de una factura escaneada (OCR):
    EXTRACT("PRICES", [TextoOCR])
