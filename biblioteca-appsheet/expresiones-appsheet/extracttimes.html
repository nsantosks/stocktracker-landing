EXTRACTTIMES()

La función EXTRACTTIMES() analiza una cadena de texto en busca de patrones que coincidan con formatos de hora (como "HH:MM" o "HH:MM:SS"). Devuelve el resultado como una Lista de Horas (List of Time).

Es muy útil para procesar horarios de apertura, turnos o citas descritos en campos de texto libre, permitiendo estandarizar esa información en columnas de tipo Time.

Sintaxis de la expresión

EXTRACTTIMES( Texto )

Parámetros:

  • Texto: La cadena de texto donde se buscarán los valores de hora.

Ejemplos Prácticos

  • Extracción simple:
    EXTRACTTIMES("La tienda abre a las 09:00 y cierra a las 20:00.")
    Explicación: Devuelve una lista con dos valores: 09:00:00 y 20:00:00.
  • Obtener hora de inicio (primer valor):
    INDEX(EXTRACTTIMES([Horario_Texto]), 1)
    Explicación: Extrae la primera hora encontrada en la descripción para usarla como hora de inicio en una columna Time.
  • Validar formato de hora en comentarios:
    ISNOTBLANK(EXTRACTTIMES([Comentario]))
    Explicación: Devuelve TRUE si el usuario escribió alguna hora válida dentro de su comentario.
