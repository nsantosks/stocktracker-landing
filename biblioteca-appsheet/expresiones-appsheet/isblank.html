ISBLANK()

La expresión ISBLANK() es una función lógica que evalúa si un valor, columna o lista está vacío. Devuelve TRUE (Verdadero) si no hay contenido y FALSE (Falso) si contiene algún dato.

Es fundamental para ocultar secciones del formulario que no aplican (Show_If), crear condiciones de obligatoriedad dinámicas o verificar si una relación de tablas tiene registros hijos.

Sintaxis de la expresión

ISBLANK( Valor )

Parámetros:

  • Valor: Puede ser una columna de cualquier tipo (Texto, Imagen, Referencia...), una lista o el resultado de otra expresión.
Su opuesto: Existe también la función ISNOTBLANK(), que es exactamente lo contrario.
Usar NOT(ISBLANK([Columna])) es lo mismo que usar ISNOTBLANK([Columna]).

Ejemplos Prácticos

  • Ocultar campo "Otros detalles" si no se elige "Otros" (Show_If):
    ISNOTBLANK([Tipo_Problema])
    O lógica inversa: Si quieres que algo aparezca solo cuando un campo está vacío: ISBLANK([Fecha_Cierre]) (Ideal para mostrar botones de acción en tareas pendientes).
  • Verificar si se subió una foto:
    IF(ISBLANK([Foto_Evidencia]), "Sin Foto", "Foto Adjunta")
  • Saber si un cliente tiene pedidos (Listas):
    ISBLANK([Related Pedidos])
    Resultado: Devuelve TRUE si la lista de pedidos relacionados está vacía (el cliente es nuevo o inactivo).
