ISNOTBLANK()

La expresión ISNOTBLANK() comprueba si una columna, celda o lista tiene contenido. Devuelve TRUE si hay algún valor presente y FALSE si está vacío.

Es la forma más limpia y directa de filtrar registros incompletos, asegurar que un campo obligatorio se haya rellenado o mostrar elementos de la interfaz solo cuando ya existe información previa.

Sintaxis de la expresión

ISNOTBLANK( Valor )

Parámetros:

  • Valor: La columna, expresión o lista que deseas evaluar.
Equivalencia: Esta función es exactamente igual a escribir NOT(ISBLANK(Valor)), pero es mucho más legible y rápida de escribir.

Ejemplos Prácticos

  • Slice para clientes con correo electrónico (Filtro):
    ISNOTBLANK([Email])
    Uso: Crea una vista donde solo aparezcan los clientes contactables por email, ignorando los que tienen ese campo vacío.
  • Mostrar fecha de finalización solo si el trabajo terminó (Show_If):
    ISNOTBLANK([Fecha_Fin])
    Uso: Puedes usar esto para mostrar u ocultar secciones enteras dependiendo de si un paso previo se ha completado.
  • Identificar Padres con Hijos (Relaciones):
    ISNOTBLANK([Related Pedidos])
    Resultado: Devuelve TRUE solo para los Clientes que han realizado al menos un pedido.
