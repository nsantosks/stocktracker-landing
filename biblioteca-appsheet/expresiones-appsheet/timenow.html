TIMENOW()

La función TIMENOW() devuelve la hora actual del sistema. A diferencia de NOW() (que incluye la fecha) o TODAY() (que es solo fecha), TIMENOW() entrega un valor de tipo Time puro (ej. "14:30:05").

Es la función estándar para establecer valores iniciales en columnas de hora de entrada/salida o para calcular duraciones en tiempo real dentro de la aplicación.

Sintaxis de la expresión

TIMENOW()

Parámetros:

  • Esta función no requiere ningún parámetro. Simplemente invócala con los paréntesis vacíos.

Ejemplos Prácticos

  • Valor Inicial (Hora de Registro):
    TIMENOW()
    Explicación: Colocado en la propiedad "Initial Value" de una columna Time, rellena el campo con la hora exacta en que se abrió el formulario.
  • Calcular duración de una tarea:
    TIMENOW() - [Hora_Inicio]
    Explicación: Si [Hora_Inicio] fue a las 14:00 y ahora son las 15:30, la resta devuelve una Duración de "01:30:00".
  • Validación de horario (Solo mañanas):
    TIMENOW() < "12:00:00"
    Explicación: Devuelve TRUE solo si la hora actual es antes del mediodía. Útil para la propiedad Valid_If.
